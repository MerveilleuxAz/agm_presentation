# Portfolio - AGM

Portefeuille personnel showcasant les projets et compétences.

## Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui

## Installation

```sh
git clone <repository-url>
cd agm_presentation
npm install
```

## Développement

```sh
npm run dev
```

Le serveur de développement démarre sur `http://localhost:8080`

## Build

```sh
npm run build
```

## Déploiement

Les fichiers construits sont dans le dossier `dist/`
